## Security Background — Worth Knowing

CoinW suffered a security breach in 2023 where approximately $13 million was taken from hot wallets. That's a real thing that happened, and it's fair to factor it in.

Their response: full reimbursement of affected users, plus a platform-wide upgrade to **Multi-Party Computation (MPC) wallet technology**. MPC wallets eliminate the single-key vulnerability that caused the original breach by distributing key shares across multiple independent secure environments.

Additional security layers in use: 2FA, cold storage, anti-phishing codes, and standard KYC/AML compliance.

Are they perfect? No exchange is. But the breach response — paying users back in full and overhauling the custody system — is about as good a response as you can get from a company in that situation.
